# 実験一覧（概要表）

## 凡例

- **実験**: アルファベット＋数字（A–Z、1–2で世代/seed を区別）
- **主要条件**: override 値や run の本質的差分
- **比較軸**: 何の効果を分離するか（1 変数対照の原則）
- **結論**: 定量結果と解釈の 1 行まとめ

---

## 進捗の流れ（時間軸）

1. **Phase 0（B/C）**: co-design の有無で効果定量化  
2. **Phase 1（F–G）**: Reach で co-design 機能確認 + トポロジー自由度の試行  
3. **Phase 2（I–L）**: 転用・比較設計是正を通じてタスク識別を実証  
4. **Phase 3（TP）**: 第3タスクで戦略ファミリーの多様性を確認
