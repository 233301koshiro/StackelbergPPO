## 現在実行中

### MuJoCo版 pusher 再学習（epoch 200 〜 999）

**状態**: バックグラウンド実行中  
**ログ**: `single_run/pusher_resume_stdout.log`  
**チェックポイント保存先**: `single_run/pusher_resume/models/`  

再開コマンド（再起動が必要になった場合）:
```bash
python3 design_opt/train.py cfg=pusher num_threads=20 enable_wandb=false \
    +restore_dir=single_run/pusher epoch=200 \
    'hydra.run.dir=single_run/pusher_resume'
```

**クラッシュ原因と対処**:
- epoch 205 で `control_action_mean` が NaN → `DiagGaussian` が ValueError
- 根本原因: `max_grad_norm=40`（PPO標準の 0.5〜5 の 8〜80 倍）
- epoch 195 で exec_R_eps が 557 → 24 に急落、大きな advantage が発生 → 勾配爆発
- 修正: `conf/config.yaml` で `max_grad_norm: 40 → 5`、NaN グラジェント検出ガードを追加
